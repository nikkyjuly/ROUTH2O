# ROUTH2O
O ROUTH2O é um ecossistema de saúde integrada que une monitoramento geográfico (GPS) com bio-rastreamento. Diferente de apps que apenas contam passos ou apenas lembram de beber água, o ROUTHH2O cria uma correlação direta e dinâmica entre o terreno percorrido e a necessidade metabólica do usuário.
